Convert HTML Characters To Strings

Wed Dec 20, 2017 in  python  Basics 

## Preliminaries
import html
## Create Text
text = 'This item costs ¥400 or £4.'
## Convert To String
html.unescape(text)
'This item costs ¥400 or £4.'

## Convert To HTML Entities
html.escape(text)
'This item costs &amp;#165;400 or &amp;#163;4.'
